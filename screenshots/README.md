# Screenshots

Place your screenshot images in this folder:

1. **login.png** - Screenshot of the login page
2. **chat.png** - Screenshot of the main chat interface showing messages
3. **userlist.png** - Screenshot showing the user list and online status indicators

## Recommended:
- Use PNG format for better quality
- Recommended size: 1200-1600px width
- Make sure screenshots show the key features of your app

## How to Add:
1. Take screenshots of your deployed app
2. Save them with the names above
3. Place them in this folder
4. They will automatically appear in the README

## Alternative Method:
If you prefer to use image URLs instead of local files, you can edit the README.md file and replace:
```
![Login Page](screenshots/login.png)
```
with:
```
![Login Page](https://your-image-url.com/image.png)
```
